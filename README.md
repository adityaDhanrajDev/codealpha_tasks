# 🎮 Hangman Game

A classic text-based Hangman game implemented in Python where players guess words letter by letter before running out of attempts.

## 🎯 Features

- **Multiple Word Categories** - Words randomly selected from Animals, Fruits, Countries, and Sports
- **Visual Hangman Display** - ASCII art that progressively shows the hangman with each wrong guess
- **Input Validation** - Prevents duplicate guesses, invalid characters, and non-letter inputs
- **Clear Screen Functionality** - Clean interface that updates after each guess
- **Win/Loss Tracking** - Clear feedback when you win or lose a round
- **Play Again Option** - Continue playing multiple rounds without restarting the program
- **Category Hints** - Shows the word category to help players narrow down possibilities

## 📋 Requirements

- Python 3.6 or higher
- No external dependencies required (uses only Python standard library)
